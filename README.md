# Sentiment Analysis Web Application

A Machine Learning based Sentiment Analysis Web Application built using Python, Scikit-learn, and Flask.  
The application predicts whether a given text expresses Positive, Negative, or Neutral sentiment through a simple web interface.

This project demonstrates the integration of Natural Language Processing (NLP), Machine Learning, and Web Development.

---

## Project Overview

Sentiment Analysis is a Natural Language Processing technique used to determine emotional tone from text data.  
This project trains a machine learning model using a labeled dataset and deploys it using a Flask web application with an HTML/CSS frontend.

---

## Features

- Text sentiment prediction (Positive / Negative / Neutral)
- Machine Learning model using Logistic Regression
- Text preprocessing and cleaning
- CountVectorizer for feature extraction
- Flask backend integration
- HTML and CSS frontend UI
- Console mode testing using main.py

---

## Technologies Used

- Python
- Flask
- Pandas
- Scikit-learn
- HTML5
- CSS3

Input:
I really like this product

Output:
Positive
